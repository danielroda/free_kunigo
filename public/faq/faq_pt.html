FAQ



O que é o Kunigo?

O Kunigo é uma ferramenta colaborativa criada para ajudar qualquer um a atingir seus objetivos. Basta participar do Kunigo e dar entrada em um projeto para começar a usar as ferramentas inovadoras disponíveis no site.

O Kunigo é gratuito?

O Kunigo é inteiramente gratuito. =)

Serve para que?

O Kunigo foi criado para ajudar as pessoas em seus objetivos. Dessa forma, o Kunigo serve para uma enormidade de coisas... olha só: com ele você pode criar coisas elaboradas como um projeto artístico, um livro ou um software ou coisas mais simples, como criar uma lista de compras, uma receita de bolo ou organizar uma festa. O Kunigo foi criado para ser do tamanho dos seus projetos e todos eles são úteis para o bom funcionamento do site.

A quem se destina?

O Kunigo se destina a todos aqueles que tenham qualquer tipo de projeto ou sonhos que deseja realizar. Ou seja: qualquer um =)

O que preciso para usar?

Kunigo é uma ferramenta online, dessa forma tudo que você vai precisar é um computador conectado a internet e um browser (Firefox, Chrome, Safari, Opera, IE7,IE8 - IE6 não recomendado). O Kunigo não utiliza plugins diferentes, nem precisa instalar nada. É só entrar no site e usar e quando terminar, clicar para sair. Simples como usar um e-mail online.

Como faço para usar?

É só entrar em http://kunigo.com.br e se registrar. Pronto, o sistema irá levar você para a tela inicial do Kunigo. Se você quer saber mais detalhes sobre como usar o sistema, leia nosso tutorial clicando na lâmpada abaixo.

Posso usar na minha empresa?

Você pode usar o Kunigo onde você quiser. O sistema dispõe de ferramentas versáteis para usar em casa, no trabalho, na sua empresa ou na sua escola.

O que significa "Kunigo"?

"Kunigo" é uma palavra em esperanto. Para quem não sabe, esperanto é uma linguagem criada para ser uma linguagem única entre todos os povos. "Kunigo" significa "aliança".

Tem alguma empresa por trás disso?

O Kunigo não está vinculado a nenhuma empresa, organização ou patrocinador. Portanto, fique tranquilo que seus projetos não estão nas mãos de ninguém.

Vou perder minhas ideias/Posso vender alguma coisa?

Todo projeto definido como privado é SEU! todo projeto público está submetido a nossa licença pública.

Meus dados estão seguros?

Não enviamos um 'a' para você sem que ele esteja embaralhado no meio de dezenas de outras letras, ou seja, os dados trocados são criptografados. O banco de dados utilizado é um dos mais modernos do mercado e está hospedado em servidores profissionais da Amazon(EC2). O website está hospedado na Califórnia-EUA em uma das melhores empresas de hospedagem. Temos backups feitos a cada 3h e backup total a cada 10 dias. Afinal os seus dados são muuuuito importantes!

E a minha privacidade?

Termos de privacidade são ridículos! O mínimo de consideração, respeito e responsabilidade por parte de quem mantém um website é respeitar seus visitantes. Por tanto damos nossa palavra que faremos o possível para que ninguém bote suas garras sujas em seus dados.

Quando entrou no ar?

A primeira versão entrou no ar dia 01/02/2009, mas o site atraiu um volume tão grande de usuários em poucas horas que achamos mais responsável de nossa parte transformar o site de acordo com essas novas exigências. A versão atual entrou no ar dia 30/03/2010.
